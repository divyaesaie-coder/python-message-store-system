# python-message-store-system
A simple Python message storage system that allows users to send messages, view chat history, and filter messages by username with timestamps.
# Python Message Store System

A beginner-friendly Python project that stores and manages chat messages with timestamps.

## Features
- Send messages
- Store chat history
- Filter messages by username
- Display message timestamps
- Menu-driven console application

## Technologies Used
- Python 3
- time module

## How It Works
The program:
1. Takes username and message input
2. Stores messages with current time
3. Displays complete chat history
4. Filters messages by specific user
5. Exits the application

## Concepts Used
- Lists
- Dictionaries
- Loops
- time.strftime()
- User input handling

```bash
